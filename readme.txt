# BlogEntrega

Proyecto hecho en Django para la entrega de clase.  
Es una web tipo blog con formularios y búsqueda en base de datos.

## Qué hace
- Muestra posts creados por autores y categorías.
- Permite cargar autores, categorías y posts desde formularios.
- Tiene un buscador para encontrar posts por título o contendo.
- Usa herencia de plantillas HTML (base + las demás).
